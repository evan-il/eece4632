Hi this is FPG-AR's final submission for EECE4632 
To run our program: 
1. Download all the files of parallelismhandoff to your project directory 
2. Download the draw_cube_video.ipynb and zynq_blur_edge_combined.ipynb notebooks into the same directory
3. Draw a red circle on a piece of paper and record it
4. Upload that video into the project directory as "input.mp4". Feel free to overwrite the current input.mp4 video, that's there for your expressed convenience 
5. Run the zynq_blur_edge_combined.ipynb notebook
6. Run the draw_cube_video.ipynb notebook
7. Play the cube_output.mp4 video for the final result
